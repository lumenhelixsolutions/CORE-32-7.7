# Numobius Ladder

Canonical ladder nodes:

- Bottom rim: `4 -> 7 -> 10 -> 13 -> 4`
- Top rim: `16 -> 19 -> 22 -> 25 -> 16`
- Smooth portal rungs: `4 <-> 16`, `7 <-> 19`, `10 <-> 22`, `13 <-> 25`
- Projective collapse targets: `16 -> 7`, `19 -> 10`, `22 -> 4`, `25 -> 7`

The implementation treats high energy, high volatility, or high forcing as a portal trigger. Top-rim overpressure or top-rim dwell produces projective collapse.
